# LLM Zoomcamp 2026 Homework

This repository contains my homework, notes, and experiments completed as part of the **LLM Zoomcamp 2026** by DataTalksClub.

The goal of this repository is not only to submit homework, but also to document my learning journey while building practical applications with Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), vector search, AI agents, evaluation, monitoring, and more over the duration of the course.

## Why this repository?

I'm following the **Learn in Public** philosophy. As I progress through each module, I'll be sharing:

* 📖 Module notes and key learnings
* 💻 Homework implementations
* 🧪 Additional experiments beyond the course
* 🚀 LinkedIn posts summarizing each module
* 🤝 Challenges I encountered and how I solved them

My aim is to understand not just **how** these systems work, but also **why** they are designed the way they are.
